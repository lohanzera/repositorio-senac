Subtítulo de Seção

HTML, que significa Hypertext Markup Language, é a linguagem usada para estruturar páginas na web. Você pode enfatizar o texto ou torná-lo forte com comandos simples.

Lista de Ferramentas

  • Editor de texto
  • Navegador da web
  • Conhecimento básico de HTML

Passos para se construir uma página

  1. Abrir um editor de texto
  2. Escrever código HTML
  3. Salvar o arquivo com extesão .html
  4. Abrir o arquivo no navegador
